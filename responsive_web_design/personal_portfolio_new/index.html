Where it all started

Blender is the first tool I ever used in relation to software development. It started when I became interested in modding video games.

In 2015-2016 I made several weapon packs for a game called Unturend, marking a significant upgrade in visual fidelity. I would argue it was well received, as the modpacks have received a total of over 1.1 million unique downloads, with an active player base peaking around 30,000 online players each day.
